Projects / Repositories

TaskKiller
C# / DotNet

This small project was created after countless times windows has refused to end a task when I requested.
It essentially serves as a gutted, minimal task manager with the advantage that it can end any non-protected task on your pc while the user only has to use an easy to use GUI.
One of the main things I tried to do with this project was to make the UI simple and smooth compared to a lot of my previous projects. While this seems fairly simple in concept, I initially hit a massive wall when refreshing the process table as it was taking an extreme amount of time to refresh. This led to me creating a simple custom stopwatch class to debug what exactly was causing the slowdown.
This allowed me to find a mistake in my table generation and update code, and improve it to a point where it is over 800 times faster than the original.
As of February 2022, this project is in a state where it is both completely usable, but I will continue to work on it to make it better.
If the project is considered complete. It will be given a compiled version in the releases page of the repo. As of mid 2022, I have spent a decent amount of time trying to refine the visual side of this project, which is something I rarely due to it being out of my skillset.
After this I feel I should attempt any new projects in WPF instead of forms as WPF has much more creative freedom from the restraints that the Win32 wrapper that forms employs.

osu! Music Lib Extractor
C# / DotNet

I seem to be one of the only people left that enjoys downloading music as mp3 and playing it locally on my computer instead of streaming everything.
I also happen to be a big fan of osu!, a rhythm game. This means effectively I had thousands of songs on my hdd but there was an issue, because it wasn't intended for direct playback but rather for use in a game, it lacked mp3 metadata. All 15000+ files are simply named "music.mp3".
This program gets every file, extracting the metadata from folder paths and other related files and then attaching the metadata to the mp3, renaming it appropriately.
It then puts the new file in a folder location of the user's choosing.
This project was initially created to run out of command line only, as that is simpler and quicker to develop and debug. As the software ballooned in complexity and the annoyance of copying a file path continued every debug session, I thought it'd be beneficial to not only me, but anyone else who wanted to use this program if I created an accompanying GUI.
Overall I consider this project complete in its current state. It does what it needs to do, and I am happy with the results.

JPZedternal
UnrealScript / UE3

JPZedternal is a fork of a mod for Killing Floor 2.
This was a 2-6 person project primarily headed by me and Dylan Fleming. I initially forked this mod to adjust a few values to allow me and my friends to have a bit more control over our gameplay. After spending a while looking over the code we realised that we could do so much more and from there we spent thousands of hours on this project. This is by far my biggest project to date. Over 400 commits to the repo and I cannot release the uncompiled files publicly as the amount of work we've put in leaves the project open to theft. The link above instead will leave to the steam community workshop page.
This project had its ups and downs, sometimes work would pay off and sometimes it wouldn't. In one case I spent 2 months working on a new feature, only for it to come crumbling down in the live release. Forcing a complete reversion of months of work.

DunDef-ReadyUpAnywherer
C# / DotNet

One of my simplest projects that actually saw extended use.
It's very simple but very effective.
If you're a fan of the original Dungeon Defenders then you'll know that there's times you want to do something else at the same time. The problem arises that you need to tab back into the window to ready up for each wave.
